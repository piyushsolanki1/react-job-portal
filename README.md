# Job Portal

A simple **React-based Job Portal** where users can browse available jobs, view job details, and add their own job postings.

## 🚀 Features
- View a list of available jobs
- See detailed job descriptions
- Add new job postings
- Responsive UI for desktop and mobile

## 🛠️ Tech Stack
- **Frontend:** React, React Router, Tailwind CSS
- **Data Source:** Local JSON file (Jobs.json)
